## Install

```bash
npm install
# or
Yarn install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# Testing

- start project first in dev - npm run dev then open another term and run - npm run cypress
